    2017-10-07
    Hexo, 随记

    终于用hexo+github pages搭建好了这个个人博客并且更换spfk主题，目前还有很多用法没有搞清楚，继续摸索~

    • 搭建过程中发现几个注意点：
      1、hexo init必须和hexo install挨着使用
      2、博客的微博数不能为0，不然图片加载不出来并且自动生成HelloWorld.md。
      3、直接在文件夹下改文件名字是不行的，需要在文件title里更改。
      4、编辑md时文本开头一定一定不要有空格，不然MarkDown语法无法解释。
      5、安装Hexo时，用npm install hexo-cli -g 安装失败，error:build:hightlight error.
      解决：在CMD上运行。      (若速度过慢用淘宝镜像cnpm代替npm，之后运行npm install -g cnpm –registry=https://registry.npm.taobao.org )
    • 现在存在的问题：
      1、spfk主题分类以及标签如何使用（已解决）
      2、已经删除的主题文件夹pulic下的background里面的图片明明已经删除，为什么执行一个hexo g 又全部重新生成了？？（JS的奥秘？？）
      3、hexo spfk主题中文本背景颜色如何更改？(已解决）
      4、hexo spfk主题中MarkDown语法无法解析。（已解决）
